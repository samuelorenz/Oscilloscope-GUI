# Oscilloscope GUI Documentation

## Introduction

This project is designed to provide a graphical user interface (GUI) for oscilloscope functionalities, making it easier for users to manage and visualize their data.

## Features
- User-friendly interface
- Real-time graphing capabilities
- Data logging and retrieval

## Getting Started

To get started with the Oscilloscope GUI, clone the repository and install the required dependencies.

```bash
git clone https://github.com/samuelorenz/Oscilloscope-GUI.git
cd Oscilloscope-GUI
# Install dependencies (if applicable)
```

## Usage

Run the application using the following command:

```bash
python main.py
```
